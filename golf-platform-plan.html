  • 📋 홈 / 탐색전국 골프장 목록 + 필터(지역·상태·거리)

🟢 골프장 상태 기능

  • 1골프장 목록 조회지역 필터, 상태 정렬, 카드형 UI
  • 2골프장 상세 조회기본 정보 + 현재 상태 + 최근 후기 5개
  • 3코스 상태 등록4개 지표 별점 + 한줄 코멘트 (로그인 필요)
  • 4날씨 정보 표시OpenWeather API 연동 (무료)
  • 5즐겨찾기마이페이지에서 관리

  • 1게시글 CRUD작성 / 수정 / 삭제 / 목록 / 상세

  • 4좋아요게시글 / 댓글 좋아요

🗄️ 데이터 모델 (핵심 엔티티)

GolfCourse (골프장)

id: UUID name: string // 남서울CC region: string // 경기, 강원 ... address: string holes: number // 18, 27, 36 lat: number lng: number thumbnail: string latest_status_at: datetime avg_rating: float

CourseStatus (상태 등록)

id: UUID course_id: FK user_id: FK visited_at: date // 라운딩일 fairway: 1-5 // 페어웨이 green: 1-5 // 그린 rough: 1-5 // 러프 crowd: 1-5 // 혼잡도 comment: string images: string[] likes: number

Post (커뮤니티 게시글)

id: UUID user_id: FK category: enum // review|gear|lesson|partner|free title: string content: text images: string[] course_id: FK? // 코스리뷰일 때 likes: number views: number created_at: datetime

User (사용자)

id: UUID provider: enum // kakao | google provider_id: string nickname: string avatar: string handicap: number? home_region: string? created_at: datetime post_count: number status_count: number

파트너 골프장과 협력 가능성.
신뢰도: 높음 / 비용: 인건비
🤖
Phase 3: 크롤링 자동화
골프장 공식 사이트, SNS, 날씨 API 자동 수집 파이프라인.
법적 검토 필요.
신뢰도: 중 / 비용: 서버비
🌤️
날씨 API (즉시 적용)
OpenWeatherMap 무료 플랜으로 골프장 위치 기반 날씨 자동 표시.
일 1,000 콜 무료.
신뢰도: 높음 / 비용: 무료

🏆 랭킹 시스템 구성

🌍
전체 랭킹
누적 포인트 기준 전국 순위. 상위 1% → 레전드 뱃지 자동 부여. 매월 초기화 없이 누적 (명예의 전당 개념).
📅
이달의 랭킹
이번 달 활동 포인트 기준. 매월 1일 초기화. 1~3위에게 골프장 할인권, 굿즈 등 실물 보상으로 경쟁 동기 자극.
📍
지역 랭킹
경기/강원/제주 등 지역별 탑 리뷰어. 해당 지역 골프장 파트너십과 연계해 초청 라운딩 제공 가능.
⛳
골프장 랭킹
특정 골프장에 가장 많이 방문/후기를 올린 유저. "남서울CC 최다 방문자" 뱃지 — 골프장 파트너와 연계 리워드.
