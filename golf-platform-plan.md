## 1. 서비스 컨셉 & 핵심 가치

| 기능 축 | 설명 |
|---|---|
| 📍 **골프장 상태 피드** | 코스 컨디션, 잔디 상태, 기상, 혼잡도를 실시간에 가깝게 확인. 방문자의 생생한 후기 중심. 소수점 1자리 정밀 점수 표시 |
| 🗺️ **지도 + 리스트 뷰** | 전국 골프장을 지도에서 상태별 색상 핀으로 확인. 리스트 뷰 토글로 상세 비교 가능 |
| 💬 **골퍼 커뮤니티** | 코스 리뷰, 장비 팁, 동반자 모집, 캐디 후기 등 골프 라이프 전반의 정보 공유. 대메뉴로 독립 운영 |
| 🔔 **맞춤 알림** (v2) | 즐겨찾는 골프장 상태 변화, 내 라운딩 전날 자동 리포트 수신 |
| ⭐ **신뢰도 시스템** | 직접 다녀온 유저만 상태 입력 가능. 검증된 정보 우선 노출 |

## 4. 기능 명세 — Phase 1 MVP

### 골프장 상태 기능

| # | 기능 | 상세 |
|---|---|---|
| 1 | 지도 뷰 | Leaflet + OpenStreetMap, 상태 색상 핀, 마커 클릭 상세 패널 |
| 2 | 리스트 뷰 | 카드 그리드, 소수점 점수 바, 평균 원형 배지, 토글 전환 |
| 3 | 골프장 상세 | 소수점 4개 지표 + 날씨 API + 최근 후기 5개 |
| 4 | 상태 등록 | 소수점 1자리 슬라이더 + 코멘트 + 사진 (로그인 필요) |
| 5 | 날씨 연동 | OpenWeatherMap 무료 API |
| 6 | 즐겨찾기 | 마이페이지 관리 |
| 7 | 지역 필터 + 정렬 | 상태좋은순 / 한산한순 / 최근업데이트 |

### 커뮤니티 기능 (대메뉴)

| # | 기능 | 상세 |
|---|---|---|
| 1 | 게시글 CRUD | 작성/수정/삭제/목록/상세 |
| 2 | 카테고리 | 코스리뷰 / 장비 / 레슨 / 동반자모집 / 자유 |
| 3 | 댓글 | 1depth (대댓글은 v2) |
| 4 | 좋아요 | 게시글 / 댓글 |
| 5 | 이미지 업로드 | 게시글당 최대 4장 |
| 6 | 랭킹 위젯 | 이달의 랭킹, 인기 골프장, 내 등급 현황 |

### Phase 1 제외 (v2+)

- 푸시 알림 / 라운딩 전날 자동 리포트
- 크롤링 자동화

## 6. 데이터 수집 전략

| 단계 | 방법 | 신뢰도 | 비용 | 시기 |
|---|---|---|---|---|
| Phase 1 | **유저 직접 입력** (더미 데이터로 프로토타입 시작) | 중 | 무료 | 즉시 |
| Phase 1 | **날씨 API** (OpenWeatherMap) | 높음 | 무료 (1,000콜/일) | 즉시 |
| Phase 2 | **운영자 관리** (어드민 직접 입력, 파트너 골프장 협력) | 높음 | 인건비 | 론칭 후 |
| Phase 3 | **크롤링 자동화** (공식 사이트, SNS, 리뷰 집계) | 중 | 서버비 | 법적검토 후 |

> 💡 **초기 전략:** 유저 입력 + 날씨 API로 빠른 론칭 → 데이터 패턴 분석 → 파트너 골프장 제휴로 공식 데이터 확보

## 7. 개발 로드맵

### Phase 1 — MVP (4~6주)

- 지도 뷰 (Leaflet, OpenStreetMap)
- 리스트 뷰 (소수점 점수 표시)
- 골프장 상세 + 상태 등록 폼
- 커뮤니티 대메뉴 (게시글 CRUD, 5개 카테고리)
- 소셜 로그인 (카카오/구글)
- 날씨 API 연동
- 더미 골프장 데이터 30개
- 모바일 반응형

### Phase 2 — Growth (6~8주)

- 검색 고도화 (엘라스틱서치)
- 알림 시스템 (앱 푸시)
- 라운딩 전날 자동 AI 리포트
- 어드민 관리 도구
- 유저 신뢰도 뱃지
- 등급/포인트 시스템 가시화
- 파트너 골프장 공식 페이지

### Phase 3 — Scale (12주+)

- 크롤링 자동화 파이프라인
- 골프장 예약 연동 (수수료 BM 가동)
- AI 코스 상태 예측
- 프리미엄 멤버십
- 네이티브 앱 (iOS/Android)

## 8. 기술 스택

| 영역 | 기술 | 이유 |
|---|---|---|
| 프론트엔드 | Next.js 14 + Tailwind CSS | SSR/SEO, 빠른 개발, 컴포넌트 재사용 |
| 지도 | Leaflet.js + OpenStreetMap | 무료, 커스텀 마커 자유도 높음 |
| 백엔드 | Next.js API Routes | 초기엔 올인원으로 단순화 |
| 데이터베이스 | Supabase (PostgreSQL) | 무료 티어 충분, Auth 내장, 실시간 구독 |
| 인증 | Supabase Auth | 카카오/구글 OAuth 기본 지원 |
| 파일 저장 | Supabase Storage | 이미지 업로드, CDN 기본 제공 |
| 날씨 API | OpenWeatherMap | 무료 1,000콜/일, 한국 데이터 정확 |
| 배포 | Vercel | Next.js 최적화, 무료 플랜 |

> 💡 **초기 비용 목표 $0:** Supabase 무료 + Vercel 무료 + OpenWeather 무료 플랜으로 프로토타입 전체 운영 가능

## 12. 골프장 상세페이지 UX 설계

> 골프장 클릭 시 우측에서 슬라이드 인 되는 390px 세로 패널. 지도/리스트 뷰와 나란히 3-column 레이아웃 구성.

### 레이아웃 구조

```
┌──────────────────────────────────────────────────────────┐
│ HEADER (고정, 58px)                                      │
├──────────────┬───────────────────────┬───────────────────┤
│              │                       │                   │
│  사이드바    │    지도 / 리스트      │  상세 패널        │
│  (310px)     │     (flex: 1)         │   (390px)         │
│  골프장 목록 │  OpenStreetMap        │  [캐러셀 상단]    │
│              │  or 카드 그리드       │  [게이지 + 바]    │
│              │                       │  [레이더 차트]    │
│              │                       │  [탭: 리뷰/정보/  │
│              │                       │      갤러리]      │
└──────────────┴───────────────────────┴───────────────────┘
```

상세 패널은 선택된 골프장이 없으면 width:0 (숨김), 클릭 시 width:390px 슬라이드 인 애니메이션.
커뮤니티 뷰 전환 시 자동 닫힘.

### 상단 포토 캐러셀

| 슬라이드 | 테마 | 그라디언트 |
|---|---|---|
| 1 | 파노라마 뷰 🌄 | 코스 상태 색상 기반 |
| 2 | 페어웨이 🌿 | 초록 계열 |
| 3 | 그린 ⛳ | 진초록 계열 |
| 4 | 클럽하우스 🏛️ | 청록 계열 |

- 자동 슬라이드: 3.5초 간격
- 수동 조작(화살표/도트 클릭) 시 타이머 리셋
- 코스 로고 뱃지: 캐러셀 하단 겹침 원형 (코스명 이니셜 2자, 코스 컬러)

### 점수 시각화 컴포넌트

#### SVG 원형 게이지 (종합 평점)
```
종합 평점 = (페어웨이 + 그린 + 러프) / 3
- 반지름 40px, stroke-width 10px
- ease-out cubic 1.1초 애니메이션
- 색상: 좋음(#52b788) / 보통(#f4a261) / 주의(#e63946)
```

#### 항목별 애니메이션 바
```
페어웨이  ████████░░  4.8
그린      █████████░  4.9
러프      ████████░░  4.2
혼잡도    ████░░░░░░  2.1
```
- 트랜지션: width 1.1초 cubic-bezier
- 색상 코딩: 각 점수별 scoreColor() 함수 적용

#### 4축 레이더 차트 (SVG)
```
          페어웨이
             ▲
             │
쾌적도 ◄────┼────► 그린
             │
             ▼
           러프
```
- 쾌적도 = 5.1 - 혼잡도 (역산, 높을수록 쾌적)
- 4단계 격자 (25%, 50%, 75%, 100%)
- 데이터 폴리곤 fill 반투명 + 꼭짓점 dot

### 탭 구조 (v0.5 개편)

골프장명 + 일반정보(주소·홀·파·날씨·업데이트) 하단에 3개 탭 배치.

| 탭 | 내용 |
|---|---|
| 📋 코스정보 (기본) | 골프장명 / 위치 / 홀·파 / 국가 / 날씨 / 업데이트 / 종합 평점 |
| 📊 코스상태 | ① 항목별 상태 (게이지 + 페어웨이·그린·러프·혼잡도 바차트)<br>② 코스 상태 분석 (레이더 차트)<br>③ 난이도 (네이버 평균스코어·슬로프 레이팅·코스 레이팅, 상급/중급/초급 배지) |
| 💬 리뷰 | 최근 방문자 후기 (닉네임 마스킹, 시간 표시) |
